Help

Die Optionen können im Optionsmenu oder via SMS aktiviert werden.

Ein SMS Befehl muss mit einem # beginnen, gefolgt vom festgelegten Passwort.
Befehle müssen mit einem - getrennt werden, z.B: #pwd-e.
Wenn der Befehl einen Parameter benötigt, kann dieser nach einem : (Strichpunkt) angehängt werden, z.B: #pwd-s:on.

Du kannst mehrere Befehle pro SMS ü,bermitteln.

Wenn du einen Befehl ohne Parameter übermittelst, wird die Funktion ausgeschaltet, mit #pwd-sms:-m: wird z.B. die SMS Weiterleitung deaktiviert.

Folgende Befehle können übermittelt werden:
  • #pwd OHNE OPTION: antwortet mit Beispielen jeder einzelnen Funktion.
  • -e antwortet mit der Liste der aktivierten Optionen.
  • -s:on / off aktiviert oder deaktiviert DroidAlone.
  • -m:aa@bb.com aktiviert die Weiterleitung per Mail an die angegebene Mail-Adresse. Ohne Parameter wird die Funktion deaktiviert.
  • -sms:123123 aktiviert die Weiterleitung per SMS an die angegebene Telefonnummer. Ohne Parameter wird die Funktion deaktiviert.
  • -r:reply aktiviert die automatische "Bin abwesend" Mitteilung. Ohne Parameter wird die Funktion deaktiviert.
  • -g:name antwortet wenn möglich mit einer Liste der dem Namen zugeordneten Nummern.
