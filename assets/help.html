Help

The options can be set using the options view, using the tutorial or remotely by sms command.

An sms command must begin with # plus the password.
Every command is separated by a -, ie: #pwd-e
If the command needs a parameter, the command name and the parameter are separated by : (colon) ie: #pwd-s:on

You can specify many parameters in the same sms command.

If the command requires a parameter but you don't provide it, it means that you want to disable that feature, ie: #pwd-sms:-m: disables sms forwarding.

The available commands are:
  • #pwd WITH NO OPTIONS: replies with an example of every single functionality
  • -e replies with the list of active options
  • -w replies with the current position of the phone. It provides the latitude - longitude and tries to retrieve a valid address
  • -s:on / off enables or disables forwarding
  • -m:aa@bb.com enables mail notification to the address specified. Null disables mail notifications
  • -sms:123123 enables sms notification to the number specified. Null disables sms notifications
  • -r:reply sets automatic I'm away reply message. Null disables replies
  • -g:name tries to retrieve the number(s) associated to the name and notifies them
